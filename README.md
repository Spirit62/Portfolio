# Personal Portfolio Site

## What It Is
This is my personal portfolio website designed to showcase my journey as a curious, lifelong learner. It highlights my academic background, involvement in community and social work, extra-curricular activities, and most importantly, my technical interests and personality.

## Why I Made It
I believe everyone should have a dedicated space to showcase their skills and growth effectively. Having recently learned HTML and CSS, I wanted to put theory into practice. Building a personal portfolio was the perfect project to implement my new skills while creating a functional digital home.

## How I Made It
The site is static, lightweight, and serves as an interactive introduction to who I am. It was built using:

* **HTML5** - Semantics and structural layout
* **CSS3** - Custom styling, Flexbox, Grid, and responsive design
* **JavaScript** - Core interactivity

## What I Learnt & Struggled With
### Key Takeaways:
* **Global Variables & Architecture:** I realized how critical it is to store colors, fonts, and theme values inside the `:root` pseudo-class for consistency and scalability. 
* **Dry CSS Principles:** I learned how to separate layout-specific styles from common/utility styles, creating a reusable global stylesheet that links cleanly across multiple pages.

### Challenges:
* **Responsive Design:** I heavily struggled with CSS media queries and finding a design layout that scales fluidly down to mobile screens without looking cluttered. It took a lot of trial, error, and layout troubleshooting, but I ultimately grasped how to handle responsive breakpoints effectively.
